# bear-game
游戏简介
《小熊与金色树叶》是一款简单有趣的休闲互动游戏，玩家需要帮助小熊收集并吃到金色树叶，同时可以与游戏中的角色互动，体验轻松愉快的游戏氛围。游戏画面色彩明快，操作简单易懂，适合各年龄段玩家体验。
游戏特色
可爱的角色设计与生动的动画效果
简单直观的拖拽操作方式
丰富的角色互动与对话系统
响应式设计，适配不同屏幕尺寸
多种游戏元素：金树叶收集、豆浆制作、垃圾清理等
游戏操作
收集金色树叶：拖动树上的金色树叶到小熊嘴边，让小熊进食
饮用豆浆：点击豆浆机，小熊会移动过去饮用豆浆（有冷却时间）
扔垃圾与清理：点击 "扔垃圾" 按钮可以生成垃圾，小熊会自动清理垃圾
重新开始：随时可以通过 "重新开始" 按钮重置游戏状态
设置调整：可以调整音效音量、动画速度等游戏参数
游戏界面
主游戏区：展示小熊、金叶树、豆浆机等游戏元素
饱腹度条：显示小熊当前的饱腹程度
对话框：记录游戏中的角色对话
导航栏：包含游戏控制按钮（重新开始、帮助、扔垃圾、设置、关于）
弹窗系统：包含帮助、设置和关于信息
技术实现
使用 HTML5 构建游戏结构
采用 Tailwind CSS v3 实现响应式布局和样式设计
利用 GSAP 动画库创建流畅的动画效果
使用 Font Awesome 提供图标支持
原生 JavaScript 实现游戏逻辑和交互功能
角色介绍
小熊：游戏主角，喜欢吃金色树叶和垃圾，会对玩家的操作做出反应
红豆：游戏中的另一个角色，会与小熊互动
金叶树：提供金色树叶的树
豆浆机：可以为小熊提供豆浆的道具
游戏目标
帮助小熊吃到足够的金色树叶，填满饱腹度条。过程中可以与各种游戏元素互动，体验轻松有趣的游戏过程。
